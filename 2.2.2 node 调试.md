# node 调试.


## Chrome DevTools

- console.log 就不用说了。
- debugger 不推荐使用，因为：  
  使用繁琐，需手动打点。  
  若忘记删除 debugger，还会引起性能问题。
  
- node-inspector 已经退出历史舞台。node@6.3 以后内置了一个调试器，可以结合 Chrome DevTools 使用，而且比 node-inspector 更强大。
