# node-module-inquirer 

交互式命令行用户接口的集合

-GitHub：https://github.com/SBoudrias/Inquirer.js#readme

- use

```
npm install inquirer
var inquirer = require('inquirer');
inquirer
  .prompt([
    /* Pass your questions in here */
  ])
  .then(answers => {
    // Use user feedback for... whatever!!
  });
```
